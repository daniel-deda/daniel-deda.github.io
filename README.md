This project is my portfolio website. I've developed it using the ReactJS framework, in order to familiarize myself with it.

This project is purely front-end, as I was playing around and fiddling with the settings while I was learning react.

If you want to see how it really works, you can run it youself by:

  1) Clone the repository to any folder of your choice
  2) cd to the repository and install npm by running: npm install
  3) Run the React App by then running: npm start
